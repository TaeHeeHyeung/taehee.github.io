html 파일 안에 html 넣기

html 작업시 확장자가 php, asp, jsp 등등은 인클루드가 가능하다.

기본적으로 공통 영역이 되는 부분들은 인클루드해서 사용한다. 그런데, html에서 html 파일 인클루드가 기본적으로 안된다고 알려져 있다.

물론 맞는말이긴하다. 하지만 js 파일을 불러온다면 html 로 인클루드가 가능하다.

<script type="text/javascript">
$(document).ready( function() {
 
$("#headers").load("common/header.html");  // 원하는 파일 경로를 삽입하면 된다
$("#footers").load("common/footer.html");  // 추가 인클루드를 원할 경우 이런식으로 추가하면 된다
 
});
</script>
